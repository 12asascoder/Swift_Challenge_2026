# AMBER Music Files

Drop your background music files here. Supported formats:
- `.mp3`
- `.m4a`
- `.wav`
- `.aif`

Files are played in alphabetical order on loop when the Daily Check-in screen opens.
They continue playing on the Home Hub and stop when a game session starts.

## Naming suggested:
- `01_amber_calm.mp3`
- `02_amber_flow.mp3`
- etc.
